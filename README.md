# 拾字 MVP

这是一个本地运行的汉字书写练习 MVP。

## 主要文件

- `index.html`：产品主页面。
- `hanzi-writer.min.js`：汉字笔顺渲染库。
- `data/`：Hanzi Writer 的单字笔画数据。
- `scripts/`：生成数据和验证页面的脚本。
- `generated/`：当前生成出的 8105 字覆盖率报告和 6854 个可写候选字结构化结果。
- `sources/`：生成脚本依赖的原始字表、字频和词典数据。
- `requirements.txt`：生成脚本需要的 Python 依赖。

当前练习节奏：每组 15 个字。脚本会从通用规范汉字表一级、二级、三级共 8105 字中检查 Hanzi Writer 笔画覆盖，保留可写且分组校验通过的 6854 个候选字，并给每个字生成语境主题、成人回忆难度分、规范等级和五级标签（小学、初中、高中、大学、专业）。前端默认跳过小学级别和语境待补字，优先把同一主题里的高中、大学、专业级字排在同一组里；三级字只在“偏挑战”且语境可用时进入新题池。

页面会在浏览器本地记录个人记忆状态：秒会、想起后写出、看提示写出、没写出来。后两类会更快进入复习，秒会的字会延后再出现。首页会显示今日到期复习、重点字和已记录数量，并提供“今日复习 / 练新字”两个入口。重点字可以直接点击练习，也可以进入“重点字本”查看词语、难度、卡住次数和下次复习时间。

“弱点画像”会根据本地练习记录，汇总用户容易卡住的主题、难度层、字形结构和具体重点字。画像里的主题、等级、结构和单字都可以点击，直接进入专项练习。

复习调度会综合最近反馈、卡住次数、掌握度和是否逾期来排序；“没写出来”和“看提示写出”会进入更短间隔，“秒会”会按掌握度延后。

新用户第一次点击“练新字”会先进入一组 15 字水平校准题，题目按中等到高难阶梯展开，并根据本组反馈自动设置初始出题偏好和个人难度偏移。已有练习记录的用户不会被重新打断。

每组 15 字内部会做会话内自适应：连续“秒会”会把后续新字推向更高难度，“想起后写出”维持当前难度，“看提示写出”小幅回退，连续“没写出来”才明显降级。每组结束后还可以标记“太简单、刚刚好、太难、词不自然”，这些整组反馈会持久化为个人难度偏移和语境严格度，下次打开仍然生效。

首页可以切换出题偏好：高频易忘、偏实用、偏挑战。每张题卡也可以标记题目质量：太简单、太生僻、词不自然、不想练。这些反馈不会覆盖记忆反馈，而是用于调整后续候选池和难度排序。

底部的“题库质检”是内部调参入口：可以按三种出题偏好查看下一组 15 字预览、100 字抽样、规范等级分布、难度层分布和主题分布，并在质检列表里直接标记太简单、太生僻、词不自然或不想练，快速修正候选池。

每组结束后会进入结果页：按四种反馈分组展示本组字，并标出下一次复习时间。结果页里的单字同样可以点击，直接进入专项练习。

## 本地预览

在这个目录中运行：

```bash
python3 -m http.server 8000
```

然后打开：

```text
http://127.0.0.1:8000/
```

## 重新生成题库

第一次运行前安装依赖：

```bash
python3 -m pip install -r requirements.txt
```

重新生成 `index.html` 内置题库、`generated/selected_8105_candidates.json` 和 `generated/hanzi_writer_coverage.json`：

```bash
python3 scripts/build_8105_chars.py
```

运行浏览器验证：

```bash
node scripts/verify_8105_app.js
```
